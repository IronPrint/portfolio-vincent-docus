# Portfolio Vincent Docus

Site statique HTML/CSS/JavaScript prêt à publier sur GitHub Pages.

## Mise en ligne rapide

1. Créer un dépôt GitHub public, par exemple `portfolio-vincent-docus`.
2. Envoyer tous les fichiers de ce dossier dans le dépôt.
3. Aller dans `Settings` > `Pages`.
4. Choisir `Deploy from a branch`.
5. Branch: `main`, Folder: `/ (root)`.
6. Sauvegarder.

Le site sera ensuite disponible à une URL du type :
`https://docusvincent.github.io/portfolio-vincent-docus/`
